Erik's REST Photo/File API

This API allows for easy file upload to a secure cloud server hosted on Digital Ocean. With this API you can also get a JSON snapshot of all files already uploaded. Files can also be retrieved for UI display. Why did I develop this API? A client asked for me to develop a portfolio personal site for their photography work. Instead of having to manually go into the cloud server and place photos 1 by 1 - I developed a custom API where I can just upload photos into the server via an HTML form (client portfolio site pulls photos from cloud server for display).

Tech Stack: This API is built in Python language using the Flask web framework. Total methods for API is 3. Methods are the following: getAllPhotoInfo, uploadPhoto, getTotalNumberOfPhotos.

Blank txt

Methods

/getAllPhotoInfo

No parameters. Method returns a JSON object of all photos/files uploaded onto server. Sample JavaScript HTTP request below:

                    
                        fetch('https://digitalocean-erik15.com/getAllPhotoInfo')
                            .then(response => response.json())
                            .then(data => {
                            console.log('Success: ', data);
                            })
                            .catch((error) => {
                            console.error('Error: ', error);
                        });

/uploadPhoto

HTTP POST request. Send via an HTML form. Method uploads a photo/file onto server. It is important that all attributes match in sample for HTML elements.

                    
                        <form action="http://localhost:5000/uploadPhoto" method="POST" enctype="multipart/form-data">
                            file: <input type="file" name="image_file" accept="image/*">
                            <input type="submit">
                        </form>

/getTotalNumberOfPhotos

HTTP GET request. This method returns a number which is the total number of photos/files that live within the server (or that user has uploaded).

                    
                        fetch('https://digitalocean-erik15.com/getTotalNumberOfPhotos')
                            .then(response => response.json())
                            .then(data => {
                            console.log('Total number of photos living in server: ', data);
                            })
                            .catch((error) => {
                            console.error('Error: ', error);
                        });
